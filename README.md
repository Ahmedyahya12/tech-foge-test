# TechForge FST — Site statique (HTML/CSS/JS)

## Fichiers
- `index.html` : page principale
- `styles.css` : style + responsive + mode sombre
- `data.js` : **à personnaliser** (liens, stats, événements)
- `app.js` : logique JS (filtre événements, toast, thème, .ics calendrier)

## Ouvrir
### Option 1
Double-clic sur `index.html`

### Option 2 (recommandée)
VS Code → extension **Live Server** → Go Live

### Option 3 (terminal)
```bash
python -m http.server 5500
```
Puis ouvre http://localhost:5500
