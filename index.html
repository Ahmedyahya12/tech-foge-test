À propos

Notre mission : rendre la tech accessible aux étudiants FST, avec du pratique et du sérieux.

Ateliers

Web, mobile, IA, sécurité, Git/GitHub, algo… avec des mini-projets.

Projets

Des projets qui ressemblent au monde réel : produit, UX, API, déploiement.

Communauté

Mentorat, entraide, sessions de révision, et préparation aux stages.

Pôles (Tracks)

Choisis un pôle et avance avec une roadmap claire.

Web React • Django/FastAPI
  • HTML/CSS/JS propre + accessibilité
  • React (hooks, state, routing)
  • API REST + Auth + déploiement
Data & IA Python • ML
  • Data cleaning + visualisation
  • Modèles ML (régression, classification)
  • Démo/mini-app (Streamlit/FastAPI)
Cyber Crypto • Web security
  • Notions : AES/RSA, hash, signature
  • OWASP Top 10 + tests
  • Bonnes pratiques backend
Mobile Flutter • Android
  • UI, state, navigation
  • Connexion API + stockage local
  • Publication & bonnes pratiques
